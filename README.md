Example
- type=github (only this for now)
- owner=sbcgua
- repo=mockup_loader
- file=src/zif_mockup_loader_constants.intf.abap

- `serverless deploy`
- `serverless logs -f getAbapVersionShieldJson --stage dev`

- `sls create_domain`, `sls delete_domain`
